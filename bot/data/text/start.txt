Neugram — это Open Source проект, который предоставляет функции ChatGPT в Telegram.

Доступ к боту является платным и составляет <b>200 RUB/год</b>. Здесь используется модель gpt-3.5-turbo-16k, которая поддерживает 16 тыс. слов контекста.

Исходный код вы можете найти на GitHub по ссылке: https://github.com/roomdie/neugram-bot.
